# Usage
<!-- usage -->
```sh-session
$ npm install -g @blueeast/bluerain-cli-electron
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
@blueeast/bluerain-cli-electron/2.0.0-aplha.1 darwin-x64 node-v10.8.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
