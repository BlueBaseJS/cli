# Usage
<!-- usage -->
```sh-session
$ npm install -g @bluebase/cli-expo
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
@bluebase/cli-expo/0.0.12 darwin-x64 node-v11.4.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
